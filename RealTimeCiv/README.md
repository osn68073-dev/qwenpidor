# Real-Time Civilization (RTS Civ)

🎮 **Игра в стиле Civilization 6, но в реальном времени!**

## ✨ Особенности

### 🎨 Улучшенная графика:
- **Градиентные текстуры** для каждого типа местности
- **Частицы и эффекты** при создании городов и юнитов
- **Плавные анимации** движения и выделения
- **Тени и свечение** для городов и юнитов
- **Современный UI** с полупрозрачными панелями
- **Миникарта** с отображением всей карты
- **Всплывающие подсказки** при наведении

### 🎯 Геймплей:
- **Реальное время** - все процессы происходят непрерывно
- **Процедурная генерация** карты 50x50
- **4 ресурса**: Еда 🍞, Золото 💰, Наука 🔬, Культура 🎭
- **Города** с автоматическим производством ресурсов
- **3 типа юнитов**: Воины ⚔️, Лучники 🏹, Кавалерия 🐎
- **5 технологий** с уникальными бонусами
- **Плавная камера** с перетаскиванием мышью

## 🗺️ Типы местности

| Местность | Цвет | Особенности |
|-----------|------|-------------|
| 🌾 Равнина | Зелёный | Базовая местность |
| 🌲 Лес | Тёмно-зелёный | Богат ресурсами |
| 🏔️ Горы | Коричневый | Препятствие |
| 🏜️ Пустыня | Жёлтый | Мало ресурсов |
| 🌊 Вода | Синий | Непроходима |

## 🚀 Как играть

1. **Откройте игру** в браузере (просто откройте `index.html`)
2. **Перемещайте камеру** перетаскиванием мыши
3. **Основывайте города** выбрав юнита и нажав "Основать город"
4. **Создавайте армию** для защиты и экспансии
5. **Исследуйте технологии** для получения бонусов

## 🎮 Управление

- **ЛКМ + перетаскивание** - Перемещение камеры
- **Клик по юниту** - Выделение юнита
- **Кнопки справа** - Действия (основать город, создать юнита)

## 💡 Советы

- Начинайте с исследования **Земледелия** для увеличения еды
- Создавайте несколько городов для ускорения производства
- Держите армию для защиты новых территорий
- Следите за прогрессом технологий на панели справа

## 🛠️ Технологии

Игра использует:
- **HTML5 Canvas** для рендеринга
- **Vanilla JavaScript** без зависимостей
- **CSS3** для современного UI
- **RequestAnimationFrame** для плавной анимации
