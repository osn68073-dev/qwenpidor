# 🏛️ Real-Time Civilization

Игра в стиле Civilization 6, но работающая в **реальном времени** без ходов!

## 🎮 Особенности

- **Реальное время**: Все процессы происходят непрерывно
- **Процедурная генерация**: Каждый раз новая уникальная карта
- **4 ресурса**: Еда, Золото, Наука, Культура
- **Разные типы местности**: Равнины, Леса, Горы, Вода, Пустыня
- **Города**: Основывайте новые города, развивайте существующие
- **Армия**: Создавайте воинов, лучников и кавалерию
- **Технологии**: Исследуйте технологии для получения бонусов
- **Миникарта**: Обзор всей игровой карты

## 🚀 Запуск

Откройте `index.html` в любом современном браузере или используйте локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте в браузере: `http://localhost:8080`

## 🎯 Управление

- **Перетаскивание мышью**: Перемещение камеры по карте
- **Клик по юниту**: Выбор юнита
- **Клик рядом с юнитом**: Перемещение выбранного юнита
- **Кнопки в панели**: Строительство, найм, исследования

## 🏗️ Игровая механика

### Ресурсы
- 🍞 **Еда**: Автоматически производится городами
- 💰 **Золото**: Нужно для строительства городов и найма юнитов
- 🔬 **Наука**: Требуется для исследований
- ⚔️ **Культура**: Пассивно накапливается от населения

### Города
- Производят ресурсы автоматически
- Могут создавать юнитов
- Контролируют территорию вокруг себя
- Имеют здоровье и население

### Юниты
- **Воин** (⚔️): 50💰, сила 10, скорость 2
- **Лучник** (🏹): 75💰, сила 8, скорость 2  
- **Кавалерия** (🐎): 120💰, сила 15, скорость 4

### Технологии
1. Земледелие → +100 еды
2. Добыча золота → +100 золота
3. Письменность → +50 науки
4. Философия → +50 культуры
5. Металлургия → +2 силы всем юнитам

## 💡 Советы

1. Сначала накопите золото для второго города
2. Исследуйте технологии для ускорения развития
3. Защищайте свои города армией
4. Контролируйте ресурсы на карте
5. Следите за миникартой для обзора территории

## 🛠️ Технологии

- HTML5 Canvas для рендеринга
- Vanilla JavaScript без фреймворков
- CSS3 для интерфейса
- RequestAnimationFrame для плавной анимации
