# 🎮 CivWeb MVP — 4X-стратегия в браузере

Веб-игра в реальном времени, вдохновлённая Civilization VI. Полностью клиентская, работает оффлайн.

## 🚀 Быстрый старт

```bash
cd civ-web-mvp
npm install
npm run dev
```

Игра откроется по адресу `http://localhost:3000`

## 🎯 Реализованные функции (MVP)

### ✅ Ядро игры
- **GameLoop**: Фиксированный тик 10 раз/сек (1 тик = 1 минута игрового времени)
- **TimeManager**: Пауза (Space), ускорение 1x/2x/4x (клавиши 1/2/3)
- **EventBus**: Декупленная система событий между модулями

### 🗺️ Карта
- **Гексагональная сетка**: Аксиальные координаты (q, r)
- **Процедурная генерация**: Simplex noise для высоты и влажности
- **Типы тайлов**: Вода, равнина, холмы, горы, лес, пустыня
- **Ресурсы**: Пшеница, скот, рыба, железо, уголь, золото, драгоценности
- **Туман войны**: Открывается при исследовании

### 🏙️ Города и экономика
- **Основание городов**: Поселенец создаёт город на пустом гексе
- **Рост населения**: При избытке еды
- **Производство**: Очередь зданий и юнитов
- **Здания**: Амбар, мастерская, рынок, библиотека, памятник
- **Ресурсы**: Еда, производство, золото, наука, культура

### 👥 Юниты
- **Типы**: Разведчик, поселенец, воин, строитель
- **Характеристики**: Движение, атака, защита, здоровье
- **Боевая система**: Урон = атака − защита ±20% рандом
- **Опыт и уровни**: За боевые действия

### 💾 Сохранения
- **IndexedDB**: Автосохранение в браузере
- **JSON экспорт/импорт**: Кнопки 💾 и 📂 в интерфейсе
- **Сериализация**: Все состояния сохраняются полностью

### 🎨 UI/UX
- **Верхняя панель**: Ресурсы в реальном времени
- **Нижняя панель**: Информация о выделенном, действия
- **Мини-карта**: Обзор исследованной территории
- **Тултипы**: Информация о тайлах при наведении
- **Уведомления**: Всплывающие сообщения
- **Камера**: WASD/стрелки, зум колёсиком

## ⌨️ Управление

| Действие | Клавиша |

| Пауза/Старт | `Space` |

## 📝 Архитектурные решения

### Data-Driven подход
Все сущности описываются через интерфейсы и конфигурации:
```typescript
interface Tile { coord, type, resource, yield, explored, visible }
interface UnitConfig { id, type, owner, coord, health, ... }
```

### Event-Driven коммуникация
Модули общаются через EventBus, что обеспечивает слабую связанность:
```typescript
eventBus.emit(GameEvent.RESOURCES_UPDATE, { resources });
eventBus.on(GameEvent.GAME_TICK, handler);
```

### Слои рендеринга
PixiJS контейнеры разделены по назначению:
- `gridContainer` — тайлы карты
- `unitsContainer` — юниты
- `citiesContainer` — города
- `selectionContainer` — выделение
- `fogContainer` — туман войны

## 🧪 Отладка

В консоли браузера доступны глобальные объекты:
```javascript
civGame.getState()      // Текущее состояние
civGame.getGrid()       // Карта
civGame.getEconomy()    // Экономика
civGame.getUnits()      // Юниты
civGame.getCities()     // Города
civGame.saveGame()      // Быстрое сохранение
```
