# CivWeb MVP - 4X Стратегия в Браузере

Реал-тайм 4X стратегия, вдохновлённая Civilization VI. Полностью клиентское приложение без серверной части.

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## 🎮 Геймплей

### Управление
- **WASD / Стрелки** - перемещение камеры
- **Колёсико мыши** - зум
- **ЛКМ** - выделение тайла
- **Пробел** - пауза
- **1/2/3** - скорость (1x, 2x, 4x)

### Ресурсы
- 🌾 **Еда** - рост населения городов
- ⚙️ **Производство** - строительство зданий и юнитов
- 💰 **Золото** - покупка юнитов, содержание армии
- 🔬 **Наука** - исследование технологий
- 🎭 **Культура** - гражданские модели

### Юниты
- 🏃 **Разведчик** - быстрая разведка карты
- 🚩 **Поселенец** - основание новых городов
- ⚔️ **Воин** - боевой юнит
- 🔨 **Строитель** - улучшение тайлов

### Технологии
Два дерева исследований:
1. **Наука** - технологии (земледелие, горное дело, письменность)
2. **Культура** - гражданские модели (ремесло, торговля, философия)

## 💾 Сохранения

### Автосохранение в IndexedDB
```typescript
import { saveManager } from './save/SaveManager';

// Сохранение
await saveManager.save(gameData, 'Моё сохранение');

// Загрузка
const save = await saveManager.load(saveId);

// Список всех сохранений
const saves = await saveManager.listSaves();
```

### Экспорт/Импорт JSON
- Кнопка **💾** - сохранить в IndexedDB
- Кнопка **📂** - импортировать из JSON файла
- Файлы сохраняются как `civweb_save_YYYY-MM-DD.json`

## 🗺️ Генерация мира

- **Гексагональная карта** с аксиальными координатами (q, r)
- **Procedural generation** через Simplex noise
- **Типы тайлов**: вода, равнина, холмы, горы, лес, пустыня
- **Ресурсы**: пшеница, скот, рыба, железо, уголь, золото, Gems
- **Туман войны** - открывается при исследовании

## 🤖 ИИ противников

4 типа личностей:
- **Mirный** - фокус на развитие и науку
- **Сбалансированный** - равновесие между войной и развитием
- **Агрессивный** - ранняя милитаризация и атаки
- **Научный** - приоритет технологий

FSM состояния ИИ:
1. **Explore** - разведка карты
2. **Expand** - основание новых городов
3. **Develop** - развитие экономики
4. **Militarize** - накопление армии
5. **Attack** - атака врагов

## 🏗️ Архитектура

### ECS-подобный подход
- **Данные** - чистые объекты (City, Unit, Tile)
- **Системы** - логика обработки (EconomySystem, AISystem)
- **События** - EventBus для декуплинга

### Игровой цикл
```typescript
// 10 тиков в секунду, 1 тик = 1 минута игрового времени
GameLoop.updateSystems(deltaTime) {
  TimeManager.tick()
  EconomySystem.update()
  AISystem.update()
  // ... другие системы
}
```

### Оптимизация
- Sprite batching в PixiJS
- Отрисовка только видимых тайлов
- Кеширование гекс-координат
- Асинхронная генерация карты
